# 📝 Tugas API

Proyek ini merupakan tugas sederhana untuk mengambil data dari API dan menampilkannya ke dalam aplikasi.

## ✨ Fitur

* 📥 **Mengambil data** dari API
* 📱 **Menampilkan data** ke halaman aplikasi
* 🌐 Menggunakan **HTTP Request**

## 🛠️ Teknologi

* 💙 **Flutter**
* 🎯 **Dart**
* 📦 **HTTP Package**

## 🎯 Tujuan

Memahami cara:

* 🔗 **Menghubungkan** aplikasi dengan API
* 🌐 **Mengambil data** dari internet
* 🔄 **Menampilkan data** secara dinamis

## 🔌 API yang Digunakan

Contoh API:

```bash
https://dummyjson.com/products

```

## 🚀 Cara Menjalankan

```bash
flutter pub get
flutter run

```
